# Product Overview

This is an Astro-based platform starter demonstrating Netlify's core primitives and modern web capabilities.

## Key Features

- **Server-side rendering (SSR)** with streaming support
- **Edge Functions** for geo-based routing and middleware
- **Netlify Blobs** for serverless storage
- **Image CDN** integration for optimized image delivery
- **On-demand revalidation** and stale-while-revalidate caching
- **News/content management** with PostgreSQL (Neon) database
- **Dynamic routing** with pagination and category filtering

## Target Use Cases

- University/institutional websites with news sections
- Content-heavy sites requiring database-backed dynamic content
- Applications needing edge computing and geo-routing
- Projects demonstrating Netlify platform capabilities
